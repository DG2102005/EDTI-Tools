# EDTI 火山引擎函数服务部署指南（备用）

> 此方案因 API 网关 Serverless 网关配额不足暂未使用。
> 火山引擎账户已充值，余额可供后续项目使用。

## 前置条件

- 火山引擎账号（可用扣子/豆包账号直接登录）
- 已实名认证
- 身边有代码文件 `volcengine-handler.js`

---

## 第1步：创建函数

1. 打开 https://console.volcengine.com/vefaas → 点击 **「创建函数」**

2. 填写以下信息：

   | 字段 | 值 |
   |------|-----|
   | 函数类型 | **事件函数** |
   | 函数名称 | `edti-customer-api` |
   | 运行环境 | **Node.js 20** |
   | 代码上传方式 | **本地上传代码包**（见第2步）或 **在线编辑** |

## 第2步：上传代码

### 方式A：在线编辑（推荐）

1. 在代码编辑区，**全选**默认的 index.js 内容
2. 打开本地文件 `cf-worker\volcengine-handler.js`
3. **全部复制粘贴**到在线编辑器中
4. 点击 **「保存」**

### 方式B：ZIP上传

1. 创建一个文件夹，里面放入 `index.js`（代码内容同 volcengine-handler.js）
2. 压缩为 ZIP
3. 上传到火山引擎

## 第3步：设置环境变量

在函数配置页 → **「环境变量」** → 添加以下5个变量：

| 变量名 | 值 |
|--------|-----|
| `FEISHU_APP_ID` | `cli_aa94b747c5f89bde` |
| `FEISHU_APP_SECRET` | `O63VOd6Iz5ToYWIk9Oflubw7Ocv5n8pl` |
| `BASE_TOKEN` | `BdZCbO3Dpai7zUsffmPc9oJSn5e` |
| `TABLE_PROJECT` | `tbl0eEtQAYoRyOpT` |
| `TABLE_MILESTONE` | `tblPj6f9JLGtVLaH` |

## 第4步：绑定触发器

1. 先开通 API 网关：https://console.volcengine.com/veapig
2. 创建 API 网关实例（需先提交工单提升配额）
3. 创建网关服务，获取公网域名
4. 回到 veFaaS → **「触发器」** → 创建触发器 → 选择 API 网关
5. 选择刚创建的网关实例 → 方法选 **ANY** → 认证免认证 → 确定

## 第5步：获取访问地址

创建成功后，页面上方会显示 **「访问路径」**。

**把这个URL复制下来发给我**，我帮你更新客户页面和分享链接。

---

## 验证是否成功

浏览器打开 `你的URL/api/health`，应该看到：

```json
{"ok":true,"t":1717000000000}
```

---

## 后续更新代码

如果以后需要更新函数代码：
1. 在火山引擎控制台找到函数
2. 在线编辑器中修改代码
3. 点击 **「部署」** 即可

不需要重新绑定触发器或设置环境变量。
