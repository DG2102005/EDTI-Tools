# EDTI 阿里云函数计算 (FC) 部署指南

## 前置条件

- 阿里云账号（未注册需先注册并实名认证）
- 身边有 `cf-worker\deploy-zip\index.js`

---

## 第1步：开通函数计算

1. 打开 https://fcnext.console.aliyun.com/
2. 首次使用点击 **「开通」**，按提示完成授权
3. 选择地域：推荐 **华东1（杭州）**

## 第2步：创建函数（事件函数）

1. 点击 **「创建函数」**
2. 选择 **「事件函数」**（不是 Web 函数！）
3. 填写以下信息：

   | 字段 | 值 |
   |------|-----|
   | 函数名称 | `edti-customer-api` |
   | 运行环境 | **Node.js 20** |
   | 代码上传方式 | **通过 ZIP 包上传代码** |
   | 函数代码 | 上传 `deploy-zip\index.js` 的 ZIP 压缩包 |

4. 点击 **「创建」**

## 第3步：创建 HTTP 触发器

1. 在函数详情页 → **「触发器」** 选项卡
2. 点击 **「创建触发器」**
3. 填写以下信息：

   | 字段 | 值 |
   |------|-----|
   | 触发器类型 | **HTTP 触发器** |
   | 请求方法 | 勾选 **GET** 和 **OPTIONS** |
   | 认证方式 | **无需认证** |
   | 禁用公网访问 URL | 不勾选 |

4. 点击 **「确定」**

5. 创建成功后，复制 **「公网访问地址」**，格式类似：
   ```
   https://xxx-xxx.cn-hangzhou.fcapp.run/
   ```

## 第4步：设置环境变量

1. 在函数详情页 → **「环境变量」** → 点击 **「编辑」**
2. 添加以下5个变量：

   | 变量名 | 值 |
   |--------|-----|
   | `FEISHU_APP_ID` | `cli_aa94b747c5f89bde` |
   | `FEISHU_APP_SECRET` | `O63VOd6Iz5ToYWIk9Oflubw7Ocv5n8pl` |
   | `BASE_TOKEN` | `BdZCbO3Dpai7zUsffmPc9oJSn5e` |
   | `TABLE_PROJECT` | `tbl0eEtQAYoRyOpT` |
   | `TABLE_MILESTONE` | `tblPj6f9JLGtVLaH` |

3. 点击 **「确定」**

## 第5步：部署代码

1. 在 **「代码」** 页签
2. 点击 **「部署代码」**（使代码和变量生效）
3. 点击 **「测试函数」** 验证是否运行正常

## 第6步：验证

浏览器打开 `你的URL/api/health`，应该看到：

```json
{"ok":true,"t":1717000000000}
```

**把这个URL复制下来发给我**，我帮你更新客户页面和分享链接。

---

## 后续更新代码

1. 在函数详情页 → **「代码」** 页签
2. 在线编辑器中修改代码，或重新上传 ZIP
3. 点击 **「部署代码」**

重新部署后会立即生效，URL 不变。
