## 快速开始

### 排期表系统
```powershell
cd EDTI-Project-Scheduler
npm install
npm start
# 访问 http://localhost:3000
```

### 报价助手
```powershell
cd EDTI-Quote-Assistant
pip install -r requirements.txt
python backend/app.py
# 访问 http://localhost:5000
```
