# EDTI 工具集

依顿检测技术服务有限公司的内部工具集合。

**GitHub 仓库**：https://github.com/DG2102005/EDTI-Tools

## 工具列表

| 工具 | 技术栈 | 说明 |
|------|--------|------|
| [EDTI-Project-Scheduler](./EDTI-Project-Scheduler/) | Node.js | 项目排期可视化系统 |
| [EDTI-Quote-Assistant](./EDTI-Quote-Assistant/) | Python/Flask | 智能报价助手 |

## 快速开始

### 排期表系统

```powershell
cd EDTI-Project-Scheduler
npm install
npm start
# 访问 http://localhost:3000
```

### 报价助手

```powershell
cd EDTI-Quote-Assistant
pip install -r requirements.txt
python backend/app.py
# 访问 http://localhost:5000
```
